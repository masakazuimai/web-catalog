## 日常運用

### 起動方法

#### タブレット（iPad/Android）

1. ブラウザ（Safari/Chrome）を開く
2. ブックマークから「カタログ」を選択
3. 自動で最初のページが表示される

#### PC

1. ブラウザを開く
2. URLを入力: `https://your-catalog-url.com/`
3. Enter キーを押す

### 基本操作

| 操作 | 方法 |
|------|------|
| 次のページ | 画面を左へスワイプ / 右下「Next」ボタン |
| 前のページ | 画面を右へスワイプ / 左下「Previous」ボタン |
| 特定ページへ | URLに `?pNo=5` を追加 |
| ガイド非表示 | 初回のみ表示、画面タップで消える |

### 初回ガイドについて

- **表示タイミング**: 初めて開いたときのみ
- **表示時間**: 3秒後に自動消去
- **手動で消す**: 画面をタップ
- **再表示させる**: ブラウザの履歴・キャッシュをクリア

## タブレット設置

### iPad設定（推奨）

#### 1. ガイドアクセス設定

キオスクモード（カタログ専用モード）の設定:

```
設定 → アクセシビリティ → ガイドアクセス
↓
「ガイドアクセス」をオン
↓
パスコードを設定
```

#### 2. カタログを全画面表示

```
Safari でカタログを開く
↓
共有ボタン → 「ホーム画面に追加」
↓
ホーム画面のアイコンからカタログを開く
↓
ホームボタン3回押し → ガイドアクセス開始
```

#### 3. 自動ロック防止

```
設定 → 画面表示と明るさ → 自動ロック → なし
```

### Android設定

#### 1. キオスクモード

```
設定 → セキュリティ → 画面の固定
↓
「画面の固定」をオン
↓
カタログを開く → 最近使用したアプリ → 固定アイコンをタップ
```

#### 2. スリープ防止

```
設定 → ディスプレイ → スリープ → 30分（または「なし」）
```

### 物理設置

#### タブレットスタンド選定

- **角度**: 45〜60度（見やすさ重視）
- **高さ**: テーブルから30〜40cm
- **固定**: 盗難防止ロック推奨

#### 設置場所

✅ **推奨**
- 店舗入口から見える位置
- 自然光が直接当たらない
- 電源コンセント近く

❌ **避けるべき**
- 直射日光が当たる
- 通路の邪魔になる
- 電源が遠い

## よくある質問

### Q1. ページめくりが反応しない

**A.** 以下を確認してください:

1. 画面を十分な距離（指1本分以上）スワイプしているか
2. 画面が汚れていないか
3. タッチパネルが正常に動作しているか（設定アプリで確認）

### Q2. 画像が表示されない

**A.** インターネット接続を確認:

1. Wi-Fi接続状態を確認
2. ブラウザを閉じて再度開く
3. タブレットを再起動

### Q3. 動作が重い・カクつく

**A.** 以下を試してください:

1. **ブラウザのタブを閉じる**
   - 他のタブを全て閉じる

2. **キャッシュをクリア**
   - Safari: 設定 → Safari → 履歴とWebサイトデータを消去
   - Chrome: 設定 → プライバシー → 閲覧履歴データの削除

3. **タブレットを再起動**
   - 電源ボタン長押し → 再起動

### Q4. 特定のページから始めたい

**A.** URLに `?pNo=X` を追加:

```
例: 5ページ目から開始
https://your-catalog-url.com/?pNo=5
```

ブックマークに登録すれば、いつも同じページから開始できます。

### Q5. 初回ガイドを再表示したい

**A.** ブラウザの履歴をクリア:

- **Safari**: 設定 → Safari → 履歴とWebサイトデータを消去
- **Chrome**: 設定 → プライバシー → 閲覧履歴データの削除

### Q6. オフラインで使えるか

**A.** 初回アクセス後はオフライン可能:

1. 一度全ページを開く（最初から最後まで）
2. 画像がキャッシュされる
3. 以降はオフラインでも閲覧可能

※完全保証ではないため、重要な場面では事前にテスト推奨

## 緊急対応

### 接続トラブル

#### 症状: カタログが全く開かない

**対処法**:

1. **Wi-Fi接続確認**
   ```
   設定 → Wi-Fi → 接続状態を確認
   ```

2. **URLが正しいか確認**
   ```
   ブックマークではなく、直接URLを入力して試す
   ```

3. **ブラウザキャッシュクリア**
   ```
   Safari/Chrome の設定からキャッシュを削除
   ```

4. **タブレット再起動**
   ```
   電源ボタン長押し → 再起動
   ```

### 表示トラブル

#### 症状: 一部のページだけ表示されない

**対処法**:

1. **ページをリロード**
   ```
   画面を下に引っ張る（プルダウンリフレッシュ）
   ```

2. **該当ページ番号を確認**
   ```
   下部の「X / 12」で現在ページを確認
   問題のページ番号を記録してIT担当者に報告
   ```

### 操作トラブル

#### 症状: スワイプが効かない

**対処法**:

1. **下部ボタンで操作**
   ```
   「Previous」「Next」ボタンを使用
   ```

2. **画面の汚れ確認**
   ```
   柔らかい布で画面を拭く
   ```

3. **タッチ感度テスト**
   ```
   設定アプリや他のアプリで正常にタッチできるか確認
   ```

### バッテリートラブル

#### 症状: すぐにバッテリーが切れる

**対処法**:

1. **常時充電設置**
   ```
   電源アダプターに常時接続
   ```

2. **画面明るさを調整**
   ```
   設定 → 画面表示 → 明るさを70%程度に
   ```

3. **バックグラウンドアプリを終了**
   ```
   使用していないアプリを全て終了
   ```
